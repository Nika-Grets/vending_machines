# Система мониторинга и анализа вендинговых сетей

Программа для отслеживания состояния торговых аппаратов и динамики продаж.

### Основной функционал:
REST API построен на FastAPI, обеспечивает доступ к данным о продажах и уведомлениям в реальном времени. Используются библиотеки Pandas для автоматического импорта и предобработки данных из CSV-отчетов.
Интерфейс написан на PyQt6 с использованием Matplotlib для визуализации бизнес-метрик (выручка, эффективность, статус работы).
Хранение данных в SQLite, оптимизированные запросы для получения статистики.

### Стек технологий:
Python 3, FastAPI, Pandas, PyQt6, Matplotlib, SQLite.

В репозитории есть файл импорта данных из csv-файла, веб-сервис и интерфейс.
